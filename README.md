# Описание работы скрипта (TL;DR)

Cкрипт демонстрирует простой пример анализа безопасности сети.

1. **Создаются тестовые сетевые логи** (имитация логов системы мониторинга).
2. **Анализируется активность IP-адресов** — подсчитывается количество событий и выявляются подозрительные IP с высокой активностью.
3. **Выполняется запрос к API Vulners**, чтобы получить информацию об известных уязвимостях (CVE) и их уровень риска (CVSS).
4. **Формируются три отчета:**

   * `log_analysis_report.csv` — анализ сетевой активности IP-адресов
   * `vulnerability_report.csv` — список найденных уязвимостей
   * `summary_report.csv` — общий отчет

5. **Строится график активности IP**, который визуализирует наиболее активные адреса.
